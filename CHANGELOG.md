## [0.1.0] - 2026-06-17

### Added

- Initial release of **Clowk Night**, the cosmic dark variant of the Clowk family, for Visual Studio Code and Cursor.
- Full workbench theming (editor, activity bar, side bar, status bar, tabs, panels, terminal, peek view, diffs, git decorations and more).
- TextMate `tokenColors` covering JavaScript/TypeScript, JSX/HTML, CSS/SCSS, JSON, YAML, TOML, Markdown, Shell, Dockerfile, Makefile, Ruby, Python, Go and Rust.
- Semantic highlighting (`semanticTokenColors`) for richer, language-server-aware colors.
- 16-color ANSI terminal palette tuned to the cosmic background.
